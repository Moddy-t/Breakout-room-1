What is HTML
    > It is the standard markup language for creating web pages.
What are tags
    > They are used to define the structure and content of an HTML document.
What are elements
    > It is an opening tag, the contents and the end tag
What is the difference between a tag and an element
    > Tags are used to define the structure and content while an element is made up of tags
